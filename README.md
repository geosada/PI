# PI
Very simple TensorFlow implementation of NVIDIA’s Π Model from [“Temporal Ensembling for Semi-Supervised Learning”] (https://arxiv.org/pdf/1610.02242.pdf) (ICLR 2017) on the SVHN classification task.
